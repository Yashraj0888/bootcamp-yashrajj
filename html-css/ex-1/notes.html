Thought Process

The goal of this project was to explore and implement different HTML structures with varying complexity. Hereâ€™s a summary of my thought process:

  • Solution 1 - This page was the simplest, containing an ordered list for movies and an unordered list for books. I also included an image on the side for additional visual interest. The goal was to create a basic webpage layout with clear content separation.
  • Solution 2 - The challenge here was introducing multiple navigation bars (left and right) to make the layout more complex. The floating navigation bars required attention to spacing and positioning. I used inline CSS for this solution, but realized that for larger projects, a separate CSS file would be more manageable.
  • Solution 3 - This page built on Solution 2 by adding detailed content and descriptions for the movies, books, and items in the table. I also included an 'About' and 'Contact' section to provide more context to the user. The main challenge was ensuring all sections were well-organized and readable, especially with multiple navigation bars.

Challenges Faced

Throughout the development process, I encountered the following challenges:

  • Aligning the navigation bars (left and right) properly without CSS frameworks required some trial and error with float properties.
  • Ensuring the page content was properly spaced and readable with the added complexity of multiple sections.
  • Adding detailed content while maintaining a clean, easy-to-read layout, especially in the movie and book lists.
