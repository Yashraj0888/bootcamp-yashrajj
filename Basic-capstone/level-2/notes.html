Project Overview

In this project, I created a simple weather widget using Web Components. The widget displays the weather information for a given city using the OpenWeather API. I made the widget reusable, so it could be embedded into any page and dynamically updated based on the city attribute passed to it.

Steps Taken

To create this weather widget, I followed these steps:

  • Created a custom web component using the `city-weather` tag.
  • Fetched weather data using the OpenWeather API for the given city.
  • Used the Shadow DOM to encapsulate styles and prevent CSS leaks from the rest of the page.
  • Created a form to allow users to change the city and update the displayed weather data dynamically.
  • Created multiple pages that used the same widget with different default cities.

Challenges Faced

While developing the weather widget, I faced the following challenges:

  • API Integration: Integrating the OpenWeather API for real-time weather data was a bit tricky at first due to API key management and rate limits. I made sure to properly handle errors and fallback scenarios.
  • Dynamic Attribute Updates: Handling the dynamic update of the city attribute on the widget was challenging. I had to listen for changes in the attribute and re-render the component accordingly.
  • Cross-Browser Compatibility: Ensuring that the Web Component worked across all modern browsers, especially with Shadow DOM encapsulation, took some time. I made sure to test the component on multiple browsers to ensure its reliability.
  • Form Interaction: Updating the widget based on user input required connecting a form to the custom element. I had to ensure that the form correctly updated the city attribute and the component re-rendered with new data.

Conclusion

Creating this weather widget was a great experience as it helped me understand Web Component development and how to encapsulate functionality in a reusable way. Despite facing challenges like API integration and dynamic attribute handling, I was able to resolve them and create a functional and interactive widget. The widget can be used across various web pages and is easy to update dynamically.
